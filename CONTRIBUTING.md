# Contributing to ENTROPY

## Getting Started

1. Fork the repo
2. Clone your fork
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Push and open a PR

## Development

### Frontend
The site is vanilla HTML/CSS/JS — no build step. Just open `index.html`.

### Engine
```bash
cd engine
npm install
node run.js
```

## Guidelines

- Keep it minimal — no frameworks
- Dark aesthetic only
- Test your changes locally before submitting
- One feature per PR

## Reporting Issues

Open an issue with:
- What you expected
- What happened
- Steps to reproduce

## Code Style

- Vanilla JS, no TypeScript
- 2-space indentation
- Single quotes
- Descriptive variable names
