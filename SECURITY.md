# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Email:** entropyindex@proton.me

Do not open a public issue for security vulnerabilities.

## Scope

- API key exposure
- Data injection via world-state.json
- XSS in the frontend

## Response

We will acknowledge reports within 48 hours and provide a fix timeline within 7 days.
